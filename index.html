Email Notification Service

As a SWE intern at Google, I had the opportunity to work on email notification service for an upcoming product. In short, my objective was to research, propose, and benchmark potential optimizations to this notification service. This was a very open-ended project as I wasn't given any hard specs to implement. By the end of my internship, I was able to produce documentation for two proposals, but was only able to provide complete benchmarks and code for one of the proposals.

Log Messaging System

As an intern for the Enterprise Infrastructure team, I had the opportunity to improve upon a log messaging system serving several enterprise applications. The messaging system is built using ZeroMQ and is used to transmit application logs for use with error reporting and performance metrics. A part of the project involved collaborating with the chief systems architect in order design an API which would standardize the formatting of the logs across the different applications. The last part of the project involved feeding the logs into a Solr instance and creating a full-text search application using Ember.js.

Sample Web Application

The CIPRES Science Gateway Project is working to provide researchers with access to HPC resources. The CIPRES REST API is a new interface through which applications can access these resources. As a web developer intern for the CIPRES team, I was tasked with creating a sample consumer application that would make use of the REST API (for documentation purposes). Using Ember.js and Bootstrap, I created a simple web application that allows the user to create and view HPC job submissions.
